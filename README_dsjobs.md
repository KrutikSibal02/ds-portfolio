# Data Science Job Market Analysis 2024 🎯
### Skills, Roles & What It Takes to Break In

## Overview
An end-to-end analysis of real Data Science job postings from 2024 to 
uncover what skills are actually demanded, which roles are most available, 
and what the minimum viable skillset looks like for entry level candidates.
Built using Python, Pandas, and Matplotlib.

## Questions Answered
- What are the most in-demand Data Science job titles?
- Which skills appear most frequently across DS job postings?
- Python vs R — what does the data actually say?
- What separates entry level requirements from senior level?
- What is the minimum viable skillset to get an entry level DS job?
- Which companies and locations hire the most?

## Key Findings
- Data Analyst is the most in-demand entry level DS role
- Python, SQL, and Machine Learning are the top 3 demanded skills
- Python appears 3.3x more than R in job postings — the debate is over
- Entry level minimum viable skillset: Python, SQL, Data Analysis, 
  Data Visualization, Machine Learning, Tableau, Statistics
- Senior roles shift from technical skills toward leadership and 
  domain expertise — the gap is about depth, not new tools
- New York leads in DS job postings for on-site roles

## Tools & Libraries
- Python, Pandas, NumPy
- Matplotlib, Seaborn

## Dataset
Data Science Job Postings & Skills 2024 — Kaggle (asaniczka)  
3 datasets merged: job postings, job skills, job summaries
