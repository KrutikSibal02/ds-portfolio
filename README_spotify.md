# Spotify Music Data Analysis 🎵

## Overview
An exploratory data analysis of 600,000+ Spotify tracks to uncover what 
makes songs popular on streaming platforms. Built using Python, Pandas, 
Seaborn, and Scikit-learn.

## Questions Answered
- What audio features correlate most with song popularity?
- How has music changed across decades?
- What is the sweet spot for song duration on streaming?
- Do explicit songs perform better than non-explicit?
- Can we predict song popularity from audio features alone?

## Key Findings
- Loudness and energy are the strongest predictors of popularity
- Music has become more energetic and less acoustic since the 1960s
- Songs between 2-4 minutes perform best on streaming platforms
- Explicit songs average higher popularity scores
- Random Forest model achieved moderate predictive accuracy — 
  popularity is influenced by factors beyond audio signals alone

## Tools & Libraries
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (RandomForestRegressor)

## Dataset
Spotify Dataset 1921-2020, 600k+ Tracks — Kaggle
